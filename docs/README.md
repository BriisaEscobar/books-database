# 📚 Book Reading Tracker

Personal project to track and analyze books I have read.

This project uses a MySQL database and a Python CLI application
to store books, authors, reading dates, and ratings.

## Technologies

- Python
- MySQL
- SQL

## Features

- Store books, authors and reading data
- Rate books and add comments
- Analyze reading habits
- View statistics about reading history

## Example Statistics

The program can calculate:

- Top rated books
- Books read per year
- Most read author
- Average rating

## Project Structure

book-reading-tracker

src/ → Python source code  
db/ → Database schema and SQL queries  
docs/ → Documentation
